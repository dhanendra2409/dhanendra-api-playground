# **App Name**: API Workbench

## Core Features:

- API Request Builder: Allow users to construct and send HTTP requests (GET, POST, PUT, DELETE, etc.) by configuring URL, HTTP method, headers, query parameters, and request body.
- API Response Viewer: Display API responses including status code, headers, and formatted body content (JSON, XML, HTML, text) with syntax highlighting.
- User Authentication & Data Persistence: Implement user login, registration, and secure storage of user-specific API requests and collections in the database.
- Request History & Collections: Enable users to save, retrieve, and organize their API requests into customizable collections for easy access and reusability.
- Dynamic Theme Switching: Provide options for users to switch between 'Dark', 'Light', and 'Dark Greenish Code' themes for the entire application interface.

## Style Guidelines:

- Dark Theme: Dominant dark gray backgrounds (#1A1A1A), light gray text (#E0E0E0), with a vibrant blue accent (#007BFF) for interactive elements.
- Light Theme: White or very light gray backgrounds (#FFFFFF, #F5F5F5), dark text (#333333), and a subtle blue accent (#2196F3).
- Dark Greenish Code Theme: Dark, slightly green-tinted background (#282C34) for code editor, with specific syntax highlighting colors (e.g., green for strings, yellow for numbers, blue for keywords).
- General UI Text: 'Inter' or 'Roboto' (sans-serif) for excellent readability across all themes.
- Code Display Font: 'Fira Code' or 'Source Code Pro' (monospace) for displaying API requests, responses, and code snippets, ensuring clear character distinction.
- Minimalist, line-art icons for HTTP methods, send/save actions, and navigation, ensuring clarity and consistency across different themes.
- Multi-panel layout with distinct sections for request configuration, response display, and request history/collections. Panels should be resizable for user customization.
- Subtle and quick transitions for loading states, theme changes, panel resizing, and tab switching to maintain a fluid and responsive user experience.
